# DevOps TechTask

Техническое задание по настройке **Nginx, Firewall и HTTPS**

---

##  Цель

1. Установка прокси-сервера **Nginx версии 1.29.6** в виртуальной машине
2. При открытии страницы на **80 порту** вывод статического текста:
   `Hello DevOps world!`
3. Настройка firewall:

   * разрешены порты: **80 (HTTP), 443 (HTTPS), 22 (SSH)**
   * остальные порты заблокированы

---

## ⚙️ Что реализовано

*  Установлен Nginx 1.29.6 (mainline)
*  Настроена статическая страница
*  Добавлен bash-скрипт обновления времени (`update_time.sh`)
*  Настроен HTTPS (self-signed сертификат)
*  Настроен firewall (UFW)
*  Настроено подключение к GitHub через SSH
*  Проект оформлен и задокументирован

---

## Проверка работы

### HTTP

```bash
curl http://localhost
```

Ожидаемый результат:

```
Hello DevOps world!
```

---

### HTTPS

```bash
curl -k https://localhost
```

> Используется `-k`, так как сертификат самоподписанный

---

### Проверка портов

```bash
nc -zv localhost 80   # открыт
nc -zv localhost 443  # открыт
nc -zv localhost 22   # открыт (если ssh запущен)
nc -zv localhost 81   # закрыт
```

---

## Firewall (UFW)

Разрешены:

* 22/tcp — SSH
* 80/tcp — HTTP
* 443/tcp — HTTPS

Все остальные входящие соединения запрещены.

---

## HTTPS

* Сгенерирован self-signed сертификат
* Настроен отдельный `server` блок на 443 порту
