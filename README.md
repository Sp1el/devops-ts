# DevOps TechTask

Техническое задание по настройке **Nginx, Firewall, HTTPS, backup и базовой автоматизации**.

---

## Цель

1. Установить прокси-сервер **Nginx версии 1.29.6** в виртуальной машине
2. Настроить вывод статической страницы на **80 порту**
3. Настроить **firewall**, разрешив только необходимые порты
4. Реализовать дополнительные задачи:
   - вывод времени на странице
   - HTTPS с self-signed сертификатом
   - backup конфигурации nginx по расписанию
   - запуск nginx от отдельного пользователя
   - подключение к GitHub по SSH
   - Dockerfile для контейнера nginx
---

## Что реализовано

-  Установлен **Nginx 1.29.6 (mainline)**
-  Настроена статическая страница с текстом `Hello DevOps world!`
-  Реализован вывод текущего времени через `update_time.sh`
-  Настроен **HTTPS** с self-signed сертификатом
-  Настроен **firewall (UFW)**
-  Реализован **backup конфигурации nginx**
-  Настроен запуск **backup через cron**
-  Worker-процессы nginx настроены на запуск от отдельного пользователя
-  Настроено подключение к GitHub через **SSH**
-  Написан **Dockerfile** для nginx
-  Проект оформлен и задокументирован

## Firewall (UFW)

Разрешены только следующие входящие порты:

* `22/tcp` — SSH
* `80/tcp` — HTTP
* `443/tcp` — HTTPS

Все остальные входящие подключения запрещены.

Проверка:

```bash
sudo ufw status verbose
```

## Вывод времени на странице

Для задания с выводом времени был создан bash-скрипт:

```text
nginx/update_time.sh
```

Скрипт генерирует HTML-страницу с текущим временем системы и записывает её в директорию, из которой nginx отдаёт страницу.

Запуск:

```bash
chmod +x nginx/update_time.sh
sudo ./nginx/update_time.sh
```

## Backup конфигурации nginx

Для резервного копирования настроек nginx создан скрипт:

```text
scripts/backup_nginx.sh
```

Он архивирует каталог `/etc/nginx` и создаёт файл формата:

```text
nginx_backup_YYYY-MM-DD_HH-MM-SS.tar.gz
```

Архивы сохраняются в:

```text
backups/
```

## Cron

Для автоматического запуска backup настроен cron.

Расписание:

```bash
0 1 * * 1 /home/sp1el/devops-ts/scripts/backup_nginx.sh
```

Это означает:

* каждый понедельник
* в 01:00

В имени архива используется время в **UTC**.

## Запуск nginx от отдельного пользователя

Для повышения безопасности был создан отдельный системный пользователь для nginx.

После настройки worker-процессы nginx работают не от `root`, а от отдельного пользователя, что уменьшает риски при возможной компрометации сервиса.

Проверка:

```bash
ps -eo user,pid,cmd | grep nginx
```

## Docker

Создан Dockerfile для запуска nginx в контейнере.

Сборка образа:

```bash
cd ~/devops-ts
sudo docker build -t my-nginx -f docker/Dockerfile .
```

Запуск контейнера:

```bash
sudo docker build -t my-nginx -f docker/Dockerfile .
```

Проверка:

```bash
curl http://localhost:8080
```
