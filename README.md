# DevOps Tesk Task

Техническое задание по настройке nginx, firewall

## Цель
1. Установка прокси-сервера Nginx версии 1.29.6 в виртуальной машине.
2. При открытии страницы сервера на 80 порту в браузере вывести статистический текст "Hello DevOps world!"
3. Настроить и установить любой firewall, чтобы разрешить подключение к серверу на 80 443 и 22 портах,а все остальные блокировали.

## Доп. Задачи
1. Объяснить свое желание стать DevOps инженером.
